    MatchMakeManager 

    Namespace: SynicSugar.MatchMake

    This is used like MatchMakeManager.Instance.XXX().

    Description 

    This script is Mono's Singleton attached to ConnenctManager. Drop this ConnenctManager into the scene from Packages/SynicSugar/Runtime/Prefabs/ConnectManager.
    ConnectManager has DontDestroy, so ConnectManager will not be destroyed by scene transitions. This is used for re-connection, and also needed for p2p scene.

    If this is no longer needed, we call CancelCurrentMatchMake, ConnectHub.Instance.CloseSession(CancellationTokenSource) or ConnectHub.Instance.ExitSession(CancellationTokenSource).
